もちPDF

かわいい PDF ビューア & エディタ

閲覧・結合・ページ編集・しおり・付箋まで。
Windows / Mac 対応の軽量ネイティブアプリ。

ブラウザで試す デスクトップ版を入手

無料・オープンソース ・ Windows 10+ / macOS 12+

機能

閲覧

単ページ表示、ズーム、ページ送り、しおり・付箋パネルでサクサク読めます。

ページ編集

タイル表示でドラッグ&ドロップで並べ替え、選択ページの一括削除・回転・抽出。

結合

複数 PDF をドラッグ&ドロップで順序指定して、ひとつにまとめます。

分割

N ページごと / しおり単位 / ファイルサイズごとなど、柔軟に分割。

しおり

追加・編集・削除して PDF アウトラインに書き戻し。階層構造もそのまま保持。

付箋

ページの好きな位置をクリックで付箋を配置。ドラッグで移動、本文は常時表示。

実際の画面

ブラウザ上で実際のアプリ画面を触ってみてください。
PDF を開く・並べ替える・しおりを編集する、すべてその場で動きます。

mochipdf / live demo
別タブで全画面表示

ブラウザ版はファイル選択ダイアログ・ダウンロード経由で動作します。
分割の「フォルダへ一括保存」など一部機能はデスクトップ版限定です。

ダウンロード

お使いの環境に合わせて選んでください。

Windows

Windows 版

  • MSI / NSIS インストーラ — `.pdf` 関連付け対応
  • ポータブル zip — 解凍してダブルクリックで起動。WebView2 同梱
最新リリースを開く

リリースページから .msi / .exe / _x64-portable.zip を選んでダウンロードできます。

macOS

Mac 版

  • .app バンドル — Apple Silicon / Intel 対応
  • サイズは約 4MB と軽量
最新リリースを開く

現時点では Mac 版はソースからのビルドを推奨しています。詳しくは README をご覧ください。
